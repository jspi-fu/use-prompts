# UI Design Principles
1. **Whitespace is King:** Always use ample padding (p-6, p-8) and gap (gap-4, gap-6). Avoid dense layouts. Content should breathe.
2. **Color Palette:**
   - NEVER use pure black (#000). Use Zinc/Slate-950 for dark text.
   - Use Zinc/Slate-500 for secondary text.
   - Backgrounds should be subtle off-white or very light gray (bg-zinc-50/10) to create depth.
3. **Visual Hierarchy:**
   - Headings (H1, H2) must be bold and tracking-tight.
   - Use muted colors for borders (border-zinc-200).
4. **Micro-Interactions:**
   - Buttons and cards must have subtle hover states.
   - Use `transition-all duration-300 ease-in-out` for smooth effects.
5. **Components:**
   - Prefer Shadcn UI / Radix UI components.
   - Use rounded-xl or rounded-2xl for cards to feel modern.
